## 1. Headings

Use `#` for headings. The number of `#` symbols determines the level.

# Heading 1 (H1)
## Heading 2 (H2)
### Heading 3 (H3)
#### Heading 4 (H4)
##### Heading 5 (H5)
###### Heading 6 (H6)

## 3. Lists

### Unordered List
Use `-` or `*` for bullet points:
- Item 1
- Item 2
  - Sub-item 2.1
  - Sub-item 2.2

### Ordered List
Use numbers:
1. First item
2. Second item
3. Third item
